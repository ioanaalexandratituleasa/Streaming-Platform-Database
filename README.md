# Streaming-Platform-Database
Database for a Streaming Platform 

 I created a database and its 5 tables using SQL.

 I learned hwo to define a primary key, foreign keys and also constraints for the attributes
 I created 10 interrogations: 5 interrogations which use data from at least 2 tables using junctions, 3 interrogations which use ORDER BY clause, 3 interrogations which use GROUP BY clause.
